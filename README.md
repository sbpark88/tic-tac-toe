## Getting Started
<!-- 필요에 따라 적절히 수정해 사용한다 -->

### Prerequisites

- npm
```shell
npm install npm@latest -g
```

or

```shell
brew install npm
```

### Installation
<!-- git clone 경로를 확인한다 -->

1. Clone the repo
   ```shell
   git clone https://github.com/sbpark88/project-repository.git
   ```
2. Install NPM packages
   ```shell
   npm install
   ```
3. Run the script
   ```shell
   npm run start
   ```
